# YMAP/YMT XML Duplicate Checker and List Exporter

This Python script checks for duplicates in `.xml` files and exports a list of unique items and duplicates.

## Features
- Scans all `.xml` files in the `input` directory.
- Counts and lists all unique items.
- Identifies and counts duplicate items.
- Exports the results to an `output.xml` file in the `output` directory.

## Setup
1. Clone this repository.
2. Place your `.xml` files in the `input` directory.
3. Run the script: `python ymap_ymt_checker.py`.

## Output
The output will be saved to `output/output.xml`, containing:
- A count of items for each input file.
- A list of duplicates for each input file.
- A list of all unique items across all files.
